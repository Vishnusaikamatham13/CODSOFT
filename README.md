# Calculator Project
This is a simple and stylish calculator built using HTML, CSS, and JavaScript.

## 🚀 Features
- Basic operations (Addition, Subtraction, Multiplication, Division)
- Responsive design
- Modern UI with dark theme
- Keyboard support

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## 📂 How to Run
1. Download the project
2. Open index.html in your browser
